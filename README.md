## 使用说明
创建一个动画帧类 var frameClass = new frame();接下来就可以调用其中的方法了。

## API
	创建帧：createFrame 
		frameClass.createFrame(function (data) {});

	编辑帧：editFrame
		frameClass.editFrame(function (data) {});

	拷贝帧：copyFrame
		frameClass.copyFrame(function (data) {});

	删除帧：deleteFrame
		frameClass.deleteFrame(function (data) {});

	设置CSS样式：setCSS()
	
	设置播放时间：setDT()

	设置位置：setINDEX()

	设置所在动画：setBELONGING()
